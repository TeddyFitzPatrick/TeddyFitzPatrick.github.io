Mandelbrot Generator

Made with vanilla JavaScript and the canvas HTML element (no libraries). The Mandelbrot set you are looking at was rendered by your browser in seconds. The canvas' pixels are colored according to their proximity to the Mandelbrot set, with a gradient surrounding the edge of the set. Proximity to the Mandelbrot set is determined by converting each pixel's cartesian (x, y) coordinates into equivalent complex coordinates and then testing these complex coordinates for divergence according to the function f(z) = z2 + c. The sharpness of the image is the number of divergence testing steps used at each coordinate, where smaller scales require higher sharpness for clarity. The default value, 1000, was arbitrarily selected for performance. PyTetris

A tetris clone made using pygame, a set of python modules wrapping Simple DirectMedia Layer (SDL) to create multimedia applications. I choose pygame, as opposed to a game engine like Unity or Godot, because it forced me to learn the fundamentals of rendering, user-input, and low-level game development.

PyLife

A GUI for Conway's Game of Life with adjustable update speed, grid-size, and tile coloring. Allows for both randomized and user-inputted initial game configurations. Grid cells will wrap around the edges of the grid instead of terminating. Fun for experimenting with popular designs, many of which can be found on the life wiki.
