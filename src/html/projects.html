PyTetris

A tetris clone made using pygame, a set of python modules wrapping Simple DirectMedia Layer (SDL) to create multimedia applications. I choose pygame, as opposed to a game engine like Unity or Godot, because it forced me to learn the fundamentals of rendering, user-input, and low-level game development.

Particle Life

Particle life simulation inspired by this video made by Tom Mohr. Randomly generated rules of attraction & repulsion between particles of different colors compose a lifelike simulation. Built with a menu to adjust each particle's sphere of influence, velocity half-life, and radii.
